# VIBE — Deep Work Sanctuary for Writers

VIBE is a private, professional, and modular AI workspace for creative writers. It transforms fragmented ideas into structured narratives through a connected toolkit, all while keeping your data strictly in your browser.

## The Toolkit
- **Oracle (Visualize)**: Brainstorm high-concept premises and synthesize multiple ideas into one masterpiece.
- **Plotline (Ideate)**: Architect your 3-act structure with character arcs and chapter beats.
- **Judge (Build)**: Get professional, honest feedback on your drafts with a visual scorecard.
- **Prompt Optimizer (Express)**: Master prompt engineering with an interactive teaching mode and annotated diffs.

## Key Features
- **BYOK (Bring Your Own Key)**: Support for Gemini, OpenAI, and Anthropic.
- **Story Bible**: A persistent context panel that flows automatically into every tool.
- **Project Persistence**: Export and import your entire project state as JSON.
- **Master Export**: Compile your progress into a clean Markdown snapshot.
- **Zero-Server Storage**: Your writing and keys never leave your browser.
- **Professional Aesthetic**: High-impact, neobrutalist design built for deep work.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- LocalStorage API

## Getting Started
1. Clone the repo and run `npm install`.
2. Launch the dev server with `npm run dev`.
3. Open the Settings (⚙️) in the header to add your Gemini, OpenAI, or Anthropic key.
4. Define your world in the **Story Bible** and start the workflow.
